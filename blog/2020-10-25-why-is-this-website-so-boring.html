why is this website so boring?

So, if you haven't noticed, this website is boring in comparison to other websites. I'd like to talk about why as to hopefully shed some light on the current state of our internet.

To start off, this website doesn't use any JavaScript. Not that js is bad, but without it, the user can know that I'm not running any ridiculous tracking/analytics software or, in other words, non-free code. Matter of fact, you can view the source code for this website.

Secondly, I've been spending time exploring websites served over different protocls like Gemini or Gopher. These are text-based websites that bring back the simplicity of the early internet. I hope to use HTML/CSS as a more modern version of what these protocols offer, and I plan to offer exclusive blog posts over Gemini in the future as well.

Last, but not least -- ok, least -- I really have no interest in a fancy or 'beautiful' front-end interface. I hope to get my content out and I hope whoever reads it finds it interesting without all the flashing lights and fancy JavaScript.

On that note, welcome to my website & blog!
